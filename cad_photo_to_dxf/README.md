# 扫描图片 / PDF 转可编辑 CAD v1.3.0（内部预览）

这是一个面向纸质图纸扫描件、手机照片和图片型 PDF 的半自动 CAD 转换工具。

当前版本采用“扫描底图保真 + 可编辑结构线辅助”的输出原则：原始页面渲染为 CAD `IMAGE` 底图，用于保留文字、符号、图框和全部细节；能够可靠识别并经人工复核的结构线以 `LINE` 叠加。自动矢量化不是原始 DWG 恢复，也不承诺 95% 或 98% 的全矢量重建率。

## 输入与多页 PDF

GUI 支持 JPG、JPEG、PNG、扫描 PDF 和图片型 PDF。

多页 PDF 不再要求导入时只选一页。程序会把 PDF 作为一个文档载入：

- 通过“上一页 / 下一页 / 页面选择器”逐页查看；
- 当前页可单独识别和可视化修改；
- 可批量自动识别全部页面；
- 导出时把全部页面合并到同一个 DXF/DWG；
- 每页建立独立 `PAGE-###` 纸空间布局；
- 模型空间同时按页面纵向排列，兼容不主动显示布局的 CAD 软件；
- 每页生成同名 `page-###.scan.png` 扫描底图。

PDF 页面本身已是矩形纸面，因此直接按页面载入，不再强制执行照片透视校正。照片输入仍可使用自动或手动四角校正。

## 输出

多页 PDF 默认生成：

```text
文档.dxf 或 文档.dwg
文档.page-001.scan.png
文档.page-002.scan.png
……
文档.report.json
```

扫描底图是外部链接，不嵌入 DXF/DWG。移动、复制或交付时，必须把 CAD 文件和全部 `page-###.scan.png` 放在同一目录并一起移动。

DWG 仍由本机安装的 ODA File Converter 从 DXF 转换。ODA 不随程序捆绑；转换失败时已生成的 DXF 和扫描底图会保留。

## 文字与符号

文字不再被当作需要用短横线、短竖线拼出的矢量图形。程序会扩大疑似文字区域保护范围，优先阻止文字笔画进入 `LINE`。原始文字始终保存在扫描底图中。

OCR 属于高级可选功能。OCR `TEXT` 的内容、字体、位置、尺寸和中文识别结果仍需人工校对，不属于默认保真路径。

## 可视化人工修改

旧版“人工复核图层”表格需要用户根据实体编号和数值猜测线段位置，已经替换为直接图上编辑：

- 在扫描图上点击或框选线段；
- 滚轮缩放，双击适应窗口；
- 将所选线段改到指定图层；
- 删除明显属于文字、噪点或误识别的线；
- 恢复全部已删除线段；
- 应用后立即更新识别预览和导出结果。

独立的“人工确认圆形”步骤已从正常流程删除。圆形候选仍可保留在报告中，但不会要求用户通过坐标表确认无法直观看见的候选。

## 视图保真

大图缩小时采用分级预览和面积采样，并对被平均变灰的细线做显示补偿。该处理只影响界面预览，不修改原始扫描图，也不修改导出的底图。缩到整张超大图纸时，显示器像素数量仍然有限；需要判断细节时应放大查看。

## 简化 GUI 流程

1. **导入图片 / 扫描 PDF**：多页 PDF 作为一个文档载入。
2. **照片校正（仅照片需要）**：PDF 页面直接使用原始矩形页面。
3. **自动识别结构线**：预处理自动执行；文字保护默认开启。
4. **在图纸上可视化修改**：直接选择、改层或删除错误线。
5. **模型尺寸校准（可选）**：用独立已知尺寸建立 `model_mm`。
6. **导出同一 CAD**：多页 PDF 合并到一个 DXF/DWG，并保留全部扫描底图。

普通使用时，高级阈值、断线连接、吸附、角度容差、OCR 和单独预处理默认折叠。只有排查特殊图纸时再展开。

## 坐标模式

- `pixel`：尚未校准的图像坐标；
- `paper_mm`：PDF 或已知纸张的纸面毫米；
- `model_mm`：通过已知实际尺寸独立校准后的模型坐标。

纸面毫米不是工程模型尺寸。没有独立模型尺度校准时，不得把 `paper_mm` 解释为真实施工尺寸。

## 环境

- Windows 10/11 x64；
- Python 3.11+；
- Windows GUI 和安装包为主要目标。

源码运行：

```powershell
python -m venv .venv
.\.venv\Scripts\Activate.ps1
python -m pip install --upgrade pip
python -m pip install -r requirements.txt
python main.py
```

## 当前边界

- 扫描底图可以保留页面视觉内容，但不等于将全部内容恢复为可编辑 CAD 实体；
- 自动结构线仍可能漏检、误检或被文字保护误删，必须在可视化编辑器中复核；
- 圆弧、样条、块、标注关联、字体、图层语义和设计约束仍无法可靠恢复；
- ODA 转换只改变文件格式，不恢复原始 DWG 语义；
- 没有原始 CAD 标准答案、逐页配准和明确误差指标时，不得宣称工程级精度；
- 当前仍是内部预览版本，不应作为正式发布或施工交付完成状态。
