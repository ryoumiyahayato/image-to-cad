# 多页合并与可视化复核工作说明

本改动解决四个相互关联的问题：多页只能逐页导出、人工复核依赖表格和坐标、识别预览在缩小时丢失细节、文字笔画被误当作 CAD 线段。

## 新工作流

1. 导入一张图片或 PDF 中的一页。
2. 扫描 PDF 页面直接保留原始渲染像素，不再默认执行第二次透视重采样。
3. 直接执行“识别线条”；预处理参数和模型尺寸校准收纳到“高级识别设置”。
4. 在“可视化编辑识别结果”中对照扫描底图：
   - 拖动整条线或线端点；
   - 修改线图层；
   - 拖动圆心、调整半径、删除或新增圆；
   - 拖动、修改、删除或新增文字；
   - 框选并删除多个错误实体。
5. 将当前页加入多页队列，再导入下一页重复处理。
6. 导出时，队列中的所有页面横向放入同一个 DXF/DWG 模型空间，并建立 `PAGE_001`、`PAGE_002` 等 DXF 组。

## 显示与文字策略

识别预览不再把扫描图和识别线条压成一张低分辨率预览图。扫描图保持完整分辨率，识别的 `LINE`、`CIRCLE` 和文字候选作为独立图形项叠加，因此缩放不会重新栅格化矢量实体。

文字区域过滤对高 DPI PDF 使用更严格的覆盖阈值，以减少汉字、尺寸文字和说明文字笔画进入 `LINE`。原始文字仍由无损 PNG 扫描底图保留；OCR 文字只有经过可视化校对并选择导出后才作为可编辑 `TEXT` 输出。

## 多页 DXF 结构

- 每页的几何实体保持为直接可编辑实体，不强制转换成块。
- 每页增加一个 `PAGE_FRAME` 边框和一个 `PAGE_LABEL` 标签。
- 每页实体加入独立的 DXF group，便于整页选择和移动。
- 每页扫描底图输出为 `<文件名>.page-001.scan.png` 等外部引用。
- DWG 仍由本机 ODA File Converter 从合并后的 DXF 转换。

## 验收边界

无损扫描底图可以逐像素保留程序实际渲染出的 PDF 页面，但这不等于自动恢复了 95%–98% 的原始 CAD 语义。自动矢量化精度仍需以真实图纸逐页统计漏线、错线、文字误线、圆和尺寸误差后判断。

在真实样本达到既定矢量验收指标之前，不应据此创建正式版本、Tag 或 Release。
